# ProjLink

ProjLink is a web-based project management system designed to streamline final year project supervision and submission within academic institutions. The system helps students and supervisors manage projects in a more organized, transparent, and efficient way.

This project was developed as a personal initiative to address common challenges faced during final year projects, such as disorganized submissions, poor communication, and difficulty tracking project progress.

---

## Features

- Student project submission and tracking
- Supervisor assignment and monitoring
- Organized project records for final year projects
- Role-based access for students and supervisors
- Centralized platform for managing project workflows

## Installation and Setup

Follow the steps below to run the project locally.

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/projlink.git
2. Navigate to the project directory
3. Create and activate a virtual environment:
   ```bash (rest terminal commands are on bash)
   python -m venv venv
   source venv/bin/activate
4. Install required dependencies:
   ```bash
   pip install -r requirements.txt
5. Apply database migrations:
   ```bash
   python manage.py migrate
6. Start the development server:
   ```bash
   python manage.py runserver
7. Open your browser and visit:
   http://127.0.0.1:8000/
